# Simple Content API application in nodejs

### Getting Started

```
yarn i
```

### Running the tests

```
yarn run dev
```

```
http://localhost:3001
```
### Routes

```javascript
[GET]/products?page=1
[GET]/products/id
[PUT]/products/id
[POST]/products
[DELETE]/products/id
```
